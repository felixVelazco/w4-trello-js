# Trello js 
## Descripción

Código en javascript que permite acceder a la página [Trello](https://trello.com), y editar según lo que necesites.

## Requisitos
- Git y github
- Cuenta en trello

## Pasos

1. Clonar este repositorio en tu computadora.
2. Obtener tu `key` y `token` de la página oficial de [trello](https://trello.com/app-key).
3. Crear el archivo `.env` en la raiz de tu repositorio con el siguiente contenido:
```
KEY= <Tu clave>
TOKEN= <Tu token>
```
4. Instala las dependencias con el comando `npm install`
